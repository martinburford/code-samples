# Select

This component includes all variants available for providing lists to the user. It's been built around native HTML `label` and `select` tags, so users (particularly on mobile devices) will be accustomed to how an option is selectable, as this component inherits the default behaviour of whichever device / operating system is being used to view the component. For example, iPhone / iOS shows options as a barrell roll, positioned towards the bottom of the devices screen.
